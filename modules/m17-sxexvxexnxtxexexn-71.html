Module 17 — The AI Artisan Workflow Library

A curated collection of proven workflows.

Key Insight

Proven workflows eliminate the need to build from scratch. By combining established patterns with verified frameworks, you can execute complex tasks with speed and confidence.

Why This Matters

Even with advanced knowledge, building a new workflow for every task is inefficient. A library of pre-built, validated workflows allows you to start immediately with a high-quality structure. This collection covers the most common high-value use cases: writing, analysis, planning, and decision-making.

Each workflow in this library integrates CLARITY-7™ for intent, SPARK-5™ for refinement, MIRROR-6™ for verification, and VALOR-Δ for value measurement. Using these templates ensures consistency and saves hours of design time.

Framework Foundations

Every workflow in this library is built on the core HQAIM pillars:

  • CLARITY-7™ anchors the intent.
  • SPARK-5™ drives the refinement process.
  • MIRROR-6™ provides the verification gates.
  • VALOR-Δ measures the final impact.

By using these standard components, the workflows remain compatible with any AI model and can be easily stored in TESSERA™ for continuity.

The Workflow Library

This section contains four complete, reusable workflows. Each is designed for a specific high-value task and is ready to copy, paste, and adapt.

Workflow 1 — The "Crystal Clear" Writing Workflow

Crystal Clear Writing 
ROLE: Expert Editor & Writer
CONTEXT: The user needs high-clarity written content.
PHASE 1: INTENT (CLARITY-7™)
- Purpose: [Define Goal]
- Audience: [Define Reader]
- Constraints: No jargon, short sentences, active voice.
- Format: [e.g., Article, Email, Report]
- Tone: Professional and direct.
- Input: [Draft or Notes]
- Expected Output: A polished, ready-to-publish text.

PHASE 2: DRAFTING
TASK: Draft the content based on Phase 1.

PHASE 3: REFINEMENT (SPARK-5™)
TASK: Refine the draft for Structure, Precision, Alignment, Readability, and Completeness.

PHASE 4: VERIFICATION (MIRROR-6™)
TASK: Verify Meaning, Accuracy, and Tone.

OUTPUT: Final Polish.
Usage Guide

Use for: Emails, reports, articles, or any communication requiring high clarity.

Key Benefit: Removes ambiguity and ensures the message lands correctly.

Workflow 2 — The "Deep Dive" Analysis Workflow

Deep Dive Analysis 
ROLE: Senior Analyst
CONTEXT: The user requires a detailed breakdown of a complex topic.
PHASE 1: DECONSTRUCTION
TASK: Break the [Topic] into its core components and relationships.

PHASE 2: EVALUATION (MIRROR-6™)
TASK: Check facts for Accuracy and Reliability. Identify any missing data.

PHASE 3: SYNTHESIS
TASK: Reassemble the findings into a coherent insight summary.

PHASE 4: VALUE CHECK (VALOR-Δ)
TASK: Rate the insight for Leverage and Outcome Strength.

OUTPUT: A structured analysis report with key takeaways.
Usage Guide

Use for: Research, data interpretation, or understanding complex systems.

Key Benefit: Turns raw information into structured, actionable insight.

Workflow 3 — The "Strategic Path" Planning Workflow

Strategic Path Planning 
ROLE: Strategic Planner
CONTEXT: The user needs a step-by-step plan to achieve [Goal].
PHASE 1: BACKWARD DESIGN
TASK: Start from the [Goal] and work backward to the present state. Identify key milestones.

PHASE 2: RESOURCE & CONSTRAINT CHECK
TASK: Review the plan against [Constraints] and available [Resources].

PHASE 3: ACTION STEPS
TASK: Convert milestones into specific, actionable tasks.

PHASE 4: VERIFICATION
TASK: Verify the logical flow and feasibility (Repeatability check).

OUTPUT: A sequential, actionable project plan.
Usage Guide

Use for: Project management, goal setting, or operational planning.

Key Benefit: Creates a logical path from goal to action.

Workflow 4 — The "Balanced Choice" Decision Workflow

Balanced Choice Decision 
ROLE: Decision Consultant
CONTEXT: The user must choose between [Option A] and [Option B].
PHASE 1: CRITERIA DEFINITION
TASK: Define the success criteria based on CLARITY-7™ intent.

PHASE 2: COMPARATIVE ANALYSIS
TASK: Compare options against criteria. List Pros/Cons and Risks/Rewards.

PHASE 3: SCORING (VALOR-Δ)
TASK: Score each option for Value, Alignment, and Outcome Strength.

PHASE 4: RECOMMENDATION
TASK: Recommend the best option based on the score and provide justification.

OUTPUT: A clear recommendation with supporting evidence.
Usage Guide

Use for: Strategic choices, vendor selection, or prioritizing tasks.

Key Benefit: Removes bias and grounds decisions in structured criteria.

Step-by-Step Method

How to use this library effectively:

  1. Identify the Task Type.
    Is it writing, analysis, planning, or decision-making?
  2. Select the Workflow.
    Copy the relevant prompt structure from the library.
  3. Customize the Brackets.
    Fill in the [Bracketed] details with your specific context.
  4. Run Phase 1.
    Start the interaction by establishing intent.
  5. Follow the Phases.
    Guide the AI through the remaining steps. Do not skip verification.
  6. Save the Result.
    Store the final output in TESSERA™ for future reference.

Typical Mistakes & Fixes

Skipping phases

Fix: Follow the sequence. Verification must happen before final output.

Vague inputs in brackets

Fix: Be specific. Replace [Goal] with the actual metric.

Ignoring the role

Fix: Ensure the AI adopts the defined persona (e.g., "Expert Editor").

Not adapting the workflow

Fix: Modify the specific constraints to fit your unique situation.

How to Check Your Results

Use the built-in frameworks within each workflow.

For Writing, check Readability (SPARK-5) and Tone (CLARITY-7).

For Analysis, check Accuracy (MIRROR-6) and Depth.

For Planning, check Feasibility and Resource Alignment.

For Decisions, check the VALOR-Δ score. Did the chosen option score highest?

Key Takeaways

  • A workflow library saves time and mental energy.
  • Standardized workflows ensure consistent quality.
  • Every workflow relies on clear intent (CLARITY-7).
  • Verification (MIRROR-6) is integral, not optional.
  • Value measurement (VALOR-Δ) confirms success.
  • Adapt these templates to build your own personal library.
